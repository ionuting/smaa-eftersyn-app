# Små-materiel Eftersyn App (TypeScript/React)

Aceasta este o conversie de bază a structurii și logicii aplicației Python în TypeScript folosind React și pdf-lib.

## Structură propusă
- React pentru UI
- pdf-lib pentru generare PDF
- Toată logica principală este adaptată din scriptul Python

## Pași următori
1. Instalează dependențele cu `npm install`
2. Rulează aplicația cu `npm start`
3. Completează funcționalitățile lipsă după necesități

## Fisiere principale
- `App.tsx` — componenta principală
- `pdfUtils.ts` — funcții pentru generarea PDF-ului
- `types.ts` — tipuri și constante

## Notă
Aceasta este o bază de pornire. Pentru funcționalitate completă, adaptează UI-ul și PDF-ul după cerințe.
